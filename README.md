## Task 2.

### Problem statement

После разработки уровней в вашей 2D игре, появилась следующая проблема: некоторые карты получаются с пересекающимися коридорами или неествественными пересечениями переходов, что ломает визуализацию.

Чтобы гарантировать, что уровень всегда можно нарисовать без пересечений, необходимо проверять граф соединений комнат и коридоров на планарность.

Если граф планарен - построить embedding.

### Out

Реализовать функции проверки графа на планарность и построения его укладки в случае истинности первой функции.

```
is_planar(graph) -> bool

get_planar_embedding(graph)
```
