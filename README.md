## Task 3.

### Problem statement

Вы уже научились генерировать **топологически корректные** уровни, теперь пришло время добавить в уровни логику.

В уровне появляются:
- переключатели (кнопки, рычаги)
- двери (открытые / закрытые)
- ловушки, порталы и так далее

Все правила уровня записываются в виде **логических ограничений** вида:
- Если игрок нажал красную кнопку - синяя дверь должна быть открыта
- Комната 52 не может одновременно содержать сокровище И быть заполнена ядом
- Либо дверь A открыта, либо переключатель B включен

Каждое такое правило - **дизъюнкция двух литералов**

Ваша задача - проверить, **существует ли** такое присваивание состояний всем переключателям и дверям, при котором **все** правила уровня выполняются одновременно.

Если да - вернуть одно возможное корректное условие уровня.

## Out

### Реализовать две функции:

**Проверка выполнимости 2-sat формулы, где clauses - список пар (x, y), означающие (x ∨ y)
переменные нумеруются от 1 до n, отрицательные - отрицания.**

**_Пример: (1, -2) означает (x1 ∨ ¬x2)_**
```python
is_satisfiable_2sat(clauses: list[tuple[int, int]], n: int) -> bool
```

**Возвращает одно возможное удовлетворяющее присваивание (True = переменная истинна) или None, если формула невыполнима.**
```python
solve_2sat(clauses: list[tuple[int, int]], n: int) -> list[bool] | None
```
